# release

## 描述

从内存中释放指定的变量，删除后该变量将无法再使用，直到重新声明。
  
Genscript不具备自动垃圾回收，因此需要开发者指明让Genscript释放哪个变量

## 参数

`variable_name` ：需要释放的变量

## 使用

`release:variable_name`

## 示例

```genscript
-a = 10     |: 声明变量a赋值为10
-@{a} is 10 |: 打印a的值
release:a   |: 从内存释放变量a
```

## 注意事项

**删除变量后，若再次需要使用该变量，必须重新声明。**
  
**请确保在确认不再需要变量时删除它，以避免影响其他逻辑或调试过程。**
