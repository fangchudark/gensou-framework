# Gensou框架  
  
## 项目简介  
  
Gensou框架是一个使用C#编写，开源的视觉小说游戏框架。  
  
该框架同时支持Unity和Godot引擎，能帮助开发者快速构建视觉小说游戏。
  
Gensou框架拥有专属的脚本语言——Genscript，它语法简单易学，专供零编程基础的开发者使用。通过Genscript，您可以轻松创建剧情对话、控制游戏的流程。
